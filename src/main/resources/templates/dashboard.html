UPI Offline Mesh Simulator

This application models a deferred-settlement transaction system. Under zero-connectivity environments, secure packets hop asynchronously over local peer devices using epidemic routing. Once a bridge node enters the Gateway cell range, the payment is settlement-verified at the backend ledger.
