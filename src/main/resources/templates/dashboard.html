UPI Offline Mesh Simulator

This application models a deferred-settlement transaction system. Under zero-connectivity environments, secure packets hop asynchronously over local peer devices using epidemic routing. Once a bridge node enters the Gateway cell range, the payment is settlement-verified at the backend ledger.

Protocol Execution Guide

  • 1
    Create Wallet Transaction
    Select accounts and amount. Click Sign & Inject to package the payload with hybrid crypto (RSA-2048 + AES-GCM-256) and inject it into an offline phone.
  • 2
    Propagate Over Mesh
    Click Execute Gossip Round. Phone nodes in range (connected by lines) will sync and share packet logs locally via peer broadcast.
  • 3
    Establish Uplink & Settle
    Drag the bridge node (e.g. phone-bridge) close to the Gateway Tower. Click Flush Gateway Uplink to ingest the payload to the server backend.
